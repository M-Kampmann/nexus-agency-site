# Nexus Agency Website

Landing page for [nexus-agency.net](https://nexus-agency.net) — AI-powered Salesforce consulting.

## Tech Stack

- Static HTML/CSS/JS
- Tailwind CSS (CDN)
- Inter + JetBrains Mono fonts (Google Fonts)
- No build step required

## Pages

- `index.html` — Main landing page (Hero, Problem, Solution, How It Works, Pricing, About, Blog, CTA, Footer)
- `blog.html` — Blog listing page (template, content coming soon)
- `privacy.html` — Privacy Policy
- `imprint.html` — Legal Imprint (Impressum)

## Deployment

Static site — deploy anywhere (Coolify, Netlify, Vercel, nginx, etc).

```bash
# For local preview
python3 -m http.server 8080
# or
npx serve .
```

## Design

- Dark theme with blue (#3381ff) accent color
- Mobile responsive
- Animated elements (fade-in on scroll, gradient text)
